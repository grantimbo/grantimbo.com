## Personal website version 2024
This is the 2024 edition of my personal website, a project that has been a profound learning experience for me. Throughout the process, I delved into the [Next.js (App Router)](https://nextjs.org/) and took on the challenge of learning [Typescript](https://www.typescriptlang.org/) within the React framework. Having been a long-time user of [Prismic](https://prismic.io), I decided to try out different content management systems and after careful consideration, I opted for [Sanity](https://sanity.io). Another noteworthy shift was transitioning from vanilla CSS (SCSS) to utilizing [Tailwind CSS](https://tailwindcss.com).

In summary, heres the stack of this project:

- [Next.js (App Router) with Typescript](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Sanity.io](https://www.sanity.io/)
